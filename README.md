# Davy Nguyen — Personal Portfolio

> A dark, hacker-aesthetic personal portfolio site built with pure HTML, CSS, and JavaScript.

## 🚀 Live Features

- **Animated typing hero** — cycles through roles with blinking cursor
- **Particle network background** — floating connected nodes
- **Glowing neon card layouts** — for projects, certs, and GitHub
- **Scroll-reveal animations** — content fades in as you scroll
- **GitHub stats widgets** — live stats, streak, and top languages
- **Responsive design** — mobile-friendly with a hamburger menu
- **Cursor glow effect** — subtle radial glow follows your mouse (desktop)
- **Console Easter egg** — open DevTools for a surprise 👾

## 📁 Project Structure

```
PortfolioSite/
├── index.html          ← Main entry point
├── style.css           ← All styles & dark theme
├── script.js           ← Animations & interactions
├── assets/
│   ├── profile.jpg     ← ⚠️ Add your photo here!
│   ├── badges/         ← Certificate badge images
│   └── projects/       ← Project screenshots
└── README.md
```

## 🖼️ Adding Your Profile Photo

1. Save your LinkedIn photo as `assets/profile.jpg`
2. The site will automatically detect and swap the placeholder icon

## 🌐 Deploying to GitHub Pages

1. Push this folder to a GitHub repo named `davyn1.github.io` (or any repo)
2. Go to **Settings → Pages** → Source: `main` branch, `/ (root)` folder
3. Your site will be live at `https://davyn1.github.io`
